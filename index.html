SPLC Extremist File URLs

The Southern Poverty Law Center are doing brave work documenting known hate groups and extremists as part of their Extremist Files and Hate Map projects. We are trying to identify the social media accounts and website URLs for these organizations to help study the impact that they are having on the Web.

If you have good search skills and a strong stomach please help us locate website, Twitter, Facebook, YouTube account URLs for these organizations. One way you can triangulate them is by searching first for their website to see if any social media accounts are linked from there. Or you can add the social media site name to a Google search for the organization name. But remember, not all of these organizations will have social media accounts or even a website.

You might want to search and browse in incognito mode so your browser and Google search history aren't polluted. To edit the spreadsheet please contact ehs@pobox.com.

The spreadsheet was derived from the geo-json that is behind SPLC's Hate Map. If you have any questions or comments please user our GitHub issue tracker.
